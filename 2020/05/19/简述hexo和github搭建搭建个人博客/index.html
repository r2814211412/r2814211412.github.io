前言

2020年4月（依旧处于寒假）

突发奇想（闲的发霉）的我决定零基础搭建一个博客用来存放一些
学习资料（乱七八糟的东西）

当然 存放的第一篇就是搭建博客的操作（白嫖）历程。当然我也是小白，还有很多不懂的
地方，所以这里只留下当初用到的网址

在本篇中搭建博客是基与GitHub Pages的免费空间。（必须注册一个github账号）

大致步骤

  • 首先需要获得（白嫖）一个域名
  • 其次git的安装以及github pages的设置
  • 将独立域名与 GitHub Pages 的空间绑定
  • 在github找一个喜欢的模板以及设置

可以用到的网址

  • 搭建个人博客基础操作

注：1.域名可以前往各大论坛找可白嫖域名
* 2.修改DNS地址不一定要在GoDaddy*
* 3.只进行到搭建完成独立博客即可*

  • 搭建hexo和安装模板

注：下载完git需要建立一个文件夹例如My blog，右键点击git Bash
here 再输入命令安装hexo
* 下载sublim text后，进入创建的文件夹随便点击一个有后缀文件，用sublim text打开，在这里面存放所写的东西*

  • 搭建之后如何上传
    一般都是 hexo n后打开sublim text，将写好的东西放进new的md文件里然后hexo s→
    hexo g→ hexo d 等几分钟即可上传成功
    如果上述操作后一直404则是需要重新在github账户绑定域名，解决方法
    这里不一定是source目录下，也可能是public目录下（主要看当初创建博客模板的大佬把字体标签图片存放到哪个目录下）
